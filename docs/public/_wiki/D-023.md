# embed 오늘폴더·_wiki 제외 (--include-today)

> 소스: `plans/done/2026-06-19.embed_today_설정화_계획.md §배경`

`discover_embed_files()`에서 `today = None`만 선언되고 실제 제외 로직이 없어, 오늘 작업 중인 Hot 파일이 매 embed마다 포함되던 버그를 수정하기로 했다. CLAUDE.md 설계 의도상 오늘 폴더·catalog는 docs_root 스캔에서 제외되어야 한다(Hot 파일은 직접 읽기 계층 담당). `today`를 `datetime.date.today().strftime("%Y-%m-%d")`로 채우고 필터에 `(include_today or parts[0] != today)` 조건을 추가해 기본 동작을 "오늘 폴더 제외"로 바로잡았다. 동시에 잦은 수정 문서를 즉시 시맨틱 검색에 포함하고 싶은 경우를 위해 `--include-today` CLI 플래그(store_true)로 override를 제공한다. catalog 폴더는 `include_today=True`여도 항상 제외된다.
