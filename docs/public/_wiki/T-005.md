# Claude Code 스킬 레이어

> 소스: `ARCHITECTURE.md §Claude Code 스킬`

스킬은 loregist 검색 계층의 소비(automation) 레이어다. 로그가 쌓이고 embed되면 스킬이 `loregist search`로 관련 컨텍스트를 주입해 반복 작업을 자동화하며, 로그화 → embed → 스킬 활용은 하나의 루프를 이룬다. 주요 스킬로는 `add-work`(오늘 작업문서에 업무 항목 등록), `carry-over`(전일 미진행 항목 이월), `daily-report`(슬랙 데일리 보고 생성), `daily-rollup`(전 프로젝트 할 일 통합), `docs-manage`(공통 문서 조회·갱신), `future-plan`(미래 계획 관리), `process-history`(히스토리 로그 분석), `catalog-update`(`_wiki/` 읽기전용 인덱스 생성), `wiki-update`(handbook + catalog 통합 갱신 오케스트레이터)가 있다. 로그를 기록하지 않으면 스킬이 참조할 과거 이력이 없으므로 기록이 출발점이다.
