# catalog (구조화 지식 증류 색인)

> 소스: `plans/done/2026-06-22.wiki_catalog_개념_통일_및_catalog_embed_포함.md §접근 방법`

catalog(`_catalog/`)는 LLM이 생성한 구조화 지식 증류물(T-xxx.md·D-xxx.md)이다. 기존에는 embed에서 명시적으로 제외돼 `loregist search`로 검색이 불가했으나, `embed.py`의 `_catalog` 제외 조건을 제거하고 `_catalog/` 하위 파일에 `source_kind="catalog"`를 부여해 docs_root의 일반 md(`source_kind="md"`)와 DB에서 구분되도록 통합했다. 단, `TOPICS.md`·`DECISIONS.md` 인덱스 파일은 개별 항목 내용의 중복 나열이라 검색 가치가 없어 제외한다. catalog 파일은 날짜 기반 rotate 대상이 아니므로 rotate 제외는 그대로 유지한다. 이로써 catalog는 wiki_sources(자유형 입출력)와 함께 embed·검색 파이프라인에서 동등하게 취급된다.
