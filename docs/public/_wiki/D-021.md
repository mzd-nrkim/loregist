# 프로젝트 레지스트리 외부화 (projects.toml)

> 소스: `plans/done/2026-06-16.멀티프로젝트_레지스트리외부화_후속계획.md §확정 설계 — 선언적 설정 파일로 외부화`

프로젝트 레지스트리를 `config.py` 안의 파이썬 dict에서 선언적 `projects.toml`로 외부화하기로 결정했다. 기존엔 프로젝트 추가/삭제가 코드 편집이라 비개발 맥락에서 마찰이 크고 실수 시 import 자체가 깨졌다. stdlib `tomllib`(Python 3.11, 런타임 의존성 0)로 파일을 읽어 `load_projects`가 기존과 동일한 `PROJECTS` dict shape으로 빌드하므로, 변수명·자료구조가 불변이라 기존 코드·테스트(monkeypatch PROJECTS)가 무파괴로 동작한다. 사용자 인터페이스만 "코드 편집"에서 "projects.toml 블록 추가/삭제"로 바뀐다. 기존 `projects --json/--current` 조회 명령도 PROJECTS dict를 직렬화하므로 로더 교체만으로 자동 파일 기반이 되어, 조회와 편집이 한 파일로 수렴한다. round-trip 편집용 라이브러리는 의존성 0 유지를 위해 도입하지 않는다.
