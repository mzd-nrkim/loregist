# 청킹 규칙 (split_log)

> 소스: `docs/public/log-format.md §청킹 규칙 요약`

loregist는 `split_log` 규칙으로 문서를 시맨틱 청크로 분리한다. 분리 기준은 빈 줄(`\n\n`)이며, 최소 청크 크기는 100자, 최대 청크 크기는 1500자다. 빈 줄로 구분된 단락이 100자 미만이면 다음 단락과 병합되고, 1500자를 초과하는 단락은 여러 청크로 분리된다. `##` 섹션 구분이 있으면 의미 단위로 나뉘어 검색 정확도가 올라간다. 따라서 단락은 빈 줄로 구분하고 각 단락을 100자 이상으로 작성하면 독립 청크로 인덱싱되어 검색 품질이 향상된다.
